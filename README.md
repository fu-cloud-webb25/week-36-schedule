###### Utveckling och driftsättning i molnmiljö, vecka 2 av 6

## Introduktion

Nu när vi kan utveckla och driftsätta våra Lambda-funktioner är det dags att göra dem tillgängliga för omvärlden. Den här veckan bygger vi våra första serverless API:er och lär oss hur API Gateway och Lambda samverkar för att skapa skalbara backendlösningar.

Vi fokuserar också på hur man skriver modulär och lättunderhållen kod för serverlessfunktioner. Med hjälp av middleware och ramverk som Middy automatiserar vi återkommande uppgifter, samtidigt som vi diskuterar hur AI kan användas för att analysera, förbättra och effektivisera vår kod.

## Mål för veckan:

1. Förstå hur API Gateway fungerar
2. Bygga serverless API:er med flera endpoints
3. Skriva modulär kod för Lambda-funktioner
5. Kunna skapa egna middlewares med hjälp av middy
